## 已知之阙

### 格里魔拉补之故
其四钮不可易文言，或生乱：
- NEW LESHY RUN 新莱西之行：若则现格里魔拉初选牌屏，莱西之行遂废
- NEW GRIMORA RUN 新格里魔拉之行：译则格里魔拉初选牌屏越而过之
- CONTINUE LESHY RUN 续莱西之行：异哉，此钮竟不可译，其故谁知？
- CONTINUE GRIMORA RUN 续格里魔拉之行：译则格里魔拉之行不可继

### 蔓尼菲科补之故
其三钮不可易文言：
- NEW LESHY RUN 新莱西之行：因与格里魔拉补首钮同名，故不可译，否则彼钮亦被易
- NEW MAGNIFICUS RUN 新蔓尼菲科之行：若上钮未译而此钮独译，则初选牌屏淆乱，试炼屏反显莱西之试
- New Magnificus Run 新蔓尼菲科之行（主界之钮）：因文与凯西补内钮同名，故亦不可译

### P03KCM补之故
其二钮不可易文言：
- NEW LESHY RUN 新莱西之行：因与格里魔拉补首钮同名，故不可译
- NEW P03 RUN 新P03之行：若上钮未译而此钮独译，则初选牌屏淆乱，试炼屏反显莱西之试
